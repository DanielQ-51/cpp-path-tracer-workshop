# ESAP Path Tracing Workshop
## by Daniel

Use the following command to compile:

```bash
g++ -O3 -std=c++17 -fopenmp main.cpp render.cpp -o render.exe
```

And use the following command to run:

```bash
./render.exe
```

This is a workshop for implementing a very basic naive path tracer.

Your task is to fill in the function renderPixel() in render.cpp.
